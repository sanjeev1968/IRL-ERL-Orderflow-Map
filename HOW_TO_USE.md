How To Use IRL ERL Orderflow Map On TradingView

This guide explains how to add the IRL ERL Orderflow Map Pine Script indicator to TradingView.

---

## Step 1: Open The Pine Script File

Open the file:

`IRL_ERL_Orderflow_Map.pine`

Copy the full Pine Script code from that file.

---

## Step 2: Open TradingView

Go to TradingView and open any chart.

At the bottom of the chart, open:

`Pine Editor`

---

## Step 3: Paste The Code

Delete any existing code inside the Pine Editor.

Paste the full code copied from:

`IRL_ERL_Orderflow_Map.pine`

---

## Step 4: Save The Script

Click:

`Save`

You can name the script:

`IRL ERL Orderflow Map`

---

## Step 5: Add To Chart

After saving, click:

`Add to chart`

The indicator will now appear on your TradingView chart.

---

## Step 6: Adjust Settings

Open the indicator settings to customize:

- Timeframes
- IRL zone visibility
- FVG / Breakaway Gap display
- Session ranges
- Previous high/low levels
- Dashboard visibility
- Colors and opacity

Use the settings according to your own trading style and timeframe preference.
